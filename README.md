# Technova - Futuristic Digital Portfolio

Technova is a high-end, futuristic digital portfolio and agency website built with Next.js, Framer Motion, and Firebase. It features a cinematic 3D carousel, dynamic language switching (Arabic/English), and a multi-step contact form integrated with Firestore.
